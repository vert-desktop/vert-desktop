Installation

Available for macOS and Linux.

# Via Homebrew
brew tap sindus/vert-desktop
brew install --cask vert-desktop

# Or download the .dmg from GitHub Releases
# Download the latest .deb
wget https://github.com/sindus/vert-desktop/releases/latest/download/vert-desktop_amd64.deb
sudo dpkg -i vert-desktop_amd64.deb
# Download the AppImage
wget https://github.com/sindus/vert-desktop/releases/latest/download/vert-desktop.AppImage
chmod +x vert-desktop.AppImage
./vert-desktop.AppImage
